Welcome to Unix Password-Manager Pages.

Basically this tool is to store your password into you local machine encrypted with a passkey. I will come into detail about the various encryption and how I decrypt it.

Highlight about the usage: 


usage: psm.py [-h] {database,insert,finder,passkey,update} ...

Personal Password Manager:

optional arguments:
  -h, --help            show this help message and exit

sub-commands:
  {database,insert,finder,passkey,update}
                        sub-command help
    database            database related commands.
    insert              insert related commands.
    finder              find details.
    passkey             passkey create or update
    update              update user details.
